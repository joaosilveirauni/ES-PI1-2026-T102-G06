# PI1-LAD.PY
Repositório para o Projeto Integrador 1 (LAD.PY).

Este projeto consiste no desenvolvimento do backend de um sistema de votação digital fictício, concebido com finalidade estritamente didática para consolidar a integração entre programação, banco de dados e fundamentos matemáticos. O sistema foi estruturado para promover a aplicação prática de conceitos de Lógica de Programação em Python, Manipulação de Bancos de Dados Relacionais via SQL e o uso de Álgebra Linear aplicada à proteção da informação. Com ênfase na segurança e confiabilidade, a ferramenta assegura a identificação adequada dos participantes e o registro consistente das escolhas realizadas, permitindo a verificação e análise rigorosa dos resultados gerados.

A aplicação funciona exclusivamente via terminal, uma escolha metodológica que prioriza o raciocínio algorítmico, a organização modular do código e a integridade referencial em detrimento de interfaces gráficas. Ao longo do desenvolvimento, são implementadas estruturas de controle para validação de documentos, modelagem de tabelas para persistência de dados e métodos de criptografia para o sigilo de informações sensíveis. Além disso, o sistema incorpora mecanismos de registro de eventos (logs) e geração de relatórios de apuração, garantindo a rastreabilidade de todas as operações. Mais do que uma solução funcional, este projeto busca demonstrar como diferentes áreas do conhecimento se articulam na construção de sistemas computacionais seguros, éticos e tecnicamente estruturados.
